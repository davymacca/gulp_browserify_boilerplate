# Gulp, Stylus, Browserify boilerplate #







** This project requires Node, Gulp and Bower **

1. [Node.js] (http://nodejs.org/)
2. [Gulp] (http://gulpjs.com/)
2. [Bower] (http://bower.io/)



### Setup ###

```shell
$ npm install
$ bower install
```




**Start the project dev**

```shell
$ gulp
```

**Build the project for release**

```shell
$ gulp --release
```
